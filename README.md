## Big O Complexity
| Big O Complexity | Recommendation | Data Type | 
| --- | --- | --- | 
| O(log n) , O(1) | Best | | 
| O(n) | Better | Iterate through a loop once | 
| O(n log n) | Bad | | 
| O(n^2) | Worse (Avoid) | Loop inside a loop | 
| O(2^n), O(n!) | Worst (Must Avoid) | |
